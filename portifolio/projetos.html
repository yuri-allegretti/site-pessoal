Experiência Criativa: Navegando na Computação

Projetos

  1. Jogo

    O primeiro projeto foi fazer um jogo usando o Construct, serviu para exercitar a lógica de programação e o trabalho em grupo. Meu grupo fez um jogo chamado "Flashing Lights" que fala sobre um mundo onde a música foi banida. O personagem principal, ao encontrar um instrumento antigo, vai em uma jornada para restaurar a música no mundo.

  2. Aplicativo Multimídia

    O segundo projeto foi criar um aplicativo com botões, textos e imagens utilizando o Processing, o que nos auxiliou a entender a introdução ao Java e criar o documento de um código.

  3. Desenvolvimento Web

    Sendo o maior desafio, o terceiro projeto fez com que tivéssemos que aprender a criar e vender uma ideia visando a solução de problemas, assim como no mercado de trabalho. Além disso, utilizamos o HTML, o CSS e o JavaScript para a programação do site, onde tivemos que desenvolver um site para a ONG chamada Crazy Cat Gang.

  4. Página Web Pessoal

    O projeto da página web pessoal acabou sendo um projeto mais fácil, pois é bem menos desafiador que o desenvolvimento web anterior. No entanto, foi mais trabalhoso e nos fez exercitar habilidades de programação em HTML, CSS e JavaScript.
